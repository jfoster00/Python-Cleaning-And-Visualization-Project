Ford GoBike User Type Analysis¶

by Justin Foster¶

Investigation Overview¶

I have looked for details regarding the Ford GoBike user types general information. In the next few slides, I will present breakdowns showing the composition of the entire user base first. These will consist of the entire user base's user type division and gender division respectively. I will then build upon those base line metrics by presenting gender composition for both the Customer user type and the Subscriber user type separately. Finally, I will end the presentation by further building on top of the previous breakdowns with to highlight average user age in years and trip time in minutes for each gender in the Customer and Subscriber user types.

Dataset Overview¶

This data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area called "Ford GoBike". Ford Gobike was created by the Ford motor company in partnership with the company "Motivate" and introduced to the San Fransisco Bay Area in 2017. The goal of the system is to help reduce traffic congestion in the city by providing an alternative mode of transportation. More information on this system can be located on Ford's media website HERE.

def uni_pie(column,title):
    # This will speed creating univariate pie charts
    sorted_counts= trips[column].value_counts()
    plt.figure(figsize=(9,7))
    plt.pie(sorted_counts,
            labels= sorted_counts.index,
            autopct='%1.0f%%',
            pctdistance= .8,
            shadow= False,
            startangle = 90,
            counterclock = False,
            wedgeprops={"edgecolor":"black",'linewidth': .5, 'antialiased': True});
    plt.title(title, fontsize= 15);

Average Trip Time for Each User Type by Gender¶

Below we begin comparing the different user types, seperated by genders, based on average number of minutes they take per trip. It is interesting to note that even though many more trips were taken by Subscriber users, it appears they have a lower average trip time in minutes in every gender category as opposed to the trips taken by Customer users.

Subscribers (least to most):

  • Male: About 9 minutes.
  • Female: About 10 minutes.
  • Other: About 11.5 minutes.

Customers (least to most):

  • Other: About 14.7 minutes.
  • Male: About 15 minutes.
  • Female: About 17 minutes.
